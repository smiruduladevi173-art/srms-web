#  SRMS-Web

Student Record Management System developed using Spring Boot, Thymeleaf, Java, JDBC, and SQLite.

## 🌐 Live Demo

https://srms-web.onrender.com/

## Features

### Student Module
- View Profile
- View Marks
- View Performance

### Staff Module
- Manage Students
- Manage Marks
- View Analytics

### Authentication
- Role-Based Login
- Session Management

## Tech Stack

- Java 21
- Spring Boot
- Thymeleaf
- SQLite
- JDBC
- HTML
- CSS
